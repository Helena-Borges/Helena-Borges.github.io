Criando seu crochê

Materiais necessários

Agulha

Existem vários tipos de tamanho de agulhas, desde as finas até as mais grossas. No início, opte por uma fina, já que com ela é mais fácil de trabalhar.

Linha

Utilize sempre a linha com espessura ideal de acordo com a agulha que irá manusear. Fique à vontade para escolher a cor!

Tesoura

A tesoura é essencial para arrematar e cortar os fios. Por isso, tenha sempre ao seu lado uma que corte bem

Diferentes tipos de pontos



Ponto Baixíssimo

Ele é um ponto para dar acabamento uma vez que é bastante discreto. Utilizando-o, é possível fazer a junção de diferentes carreiras, reforçar as bordas e também fazer a finalização das peças.


Ponto Meio Alto

Esse é um ponto intermediário entre o baixo e o alto. Ele é um pouco mais maleável que o ponto baixo, porém, bem estruturado.


Ponto Alto

O ponto alto é voltado para aquelas peças que necessitam ser mais flexíveis, por isso esse é o ponto ideal para que se dê o movimento necessário para a peça, uma vez que o mesmo é mais aberto.


Ponto Jasmine

Este design apresenta delicados pontos bufantes em forma de flor. Este ponto seria ótimo para projetos de inverno, como cachecóis, golas ou aquecedores de orelha/tiaras.



Ponto Estrela

Você está procurando um ponto de crochê elegante? Este ponto é lindo e é ótimo para elevar um padrão simples. Este é um padrão mais complicado de pegar o jeito


Ponto Scallop

A textura deste se assemelha a pequenas conchas de vieira. Este seria divertido em um xale, manta, bolsa, o que você quiser.
